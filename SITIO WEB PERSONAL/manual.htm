Manual Técnico

  1. Se abrió el Bloc de notas para escribir el código.
  2. Se creó el archivo principal llamado index.htm, que contiene un menú de navegación.
  3. Desde el menú, se enlazaron varias páginas: hoja de vida, tabla de notas, galería de imágenes, video, juego y este manual.
  4. Se elaboró una hoja de vida con información personal.
  5. Se creó una tabla de notas con los cursos actuales y sus calificaciones del primer y segundo parcial.
  6. Se diseñó una galería de imágenes con temática de la Fórmula 1 (F1), usando una tabla de imágenes.
  7. Se insertó un video de YouTube de Animal Planet usando un iframe.
  8. Se desarrolló un juego de memoria (Encuentra la pareja) utilizando HTML, CSS y JavaScript.
  9. Se aplicó estilo con CSS para personalizar colores, botones, tablas, títulos y diseño general.
  10. Finalmente, se probó el sitio web abriendo el archivo index.htm en un navegador web (doble clic).
